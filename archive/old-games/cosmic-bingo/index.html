🚀 How to Play Cosmic Bingo

🎯 Objective

Be the first to complete a winning pattern on your bingo card!

🎮 Controls

  • Desktop: Click on numbers to mark them
  • Mobile: Tap on numbers to mark them
  • Space Bar: Pause/Resume the game

🏆 Winning Patterns

  • Complete any horizontal row
  • Complete any vertical column
  • Complete either diagonal

⚡ Power-Ups

  • 🔮 Reveal: Automatically marks a random uncalled number
  • ⚡ Double: Next called number marks two cells
  • ❄️ Freeze: Freezes other players for 5 seconds
  • 🌟 Wild: Mark any number as a wildcard

👥 Multiplayer

  • Host a game to get a room code
  • Share the code with friends
  • Join games using a room code
  • Send reactions to celebrate!

🎨 Themes

Choose from 5 different visual themes to customize your experience!

📱 Tips

  • Sound effects enhance the experience
  • Your stats are saved automatically
  • Power-ups reset each game
  • The center square is always FREE
