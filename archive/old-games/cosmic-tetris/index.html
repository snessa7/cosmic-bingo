🌌 Cosmic Tetris Guide

🎮 Controls

Desktop:

  • Arrow Keys: Move and rotate pieces
  • Space: Hard drop piece
  • P: Pause game
  • R: Restart game

Mobile: Use touch controls at bottom

🎯 Game Modes

  • Classic: Endless gameplay with increasing speed
  • Sprint: Clear 40 lines as fast as possible
  • Survival: Extreme speed increases
  • Zen: Peaceful mode with no pressure

⚡ Power-ups

  • 💣 Bomb: Clears surrounding blocks
  • ⚡ Line Blaster: Clears random lines
  • 🕰️ Slow Motion: Slows down time
  • 🌟 Super Clear: Next line clear removes extra lines

🏆 Scoring

  • Single line: 100 points
  • Double line: 300 points
  • Triple line: 500 points
  • Tetris (4 lines): 800 points
  • Bonus for consecutive clears!
